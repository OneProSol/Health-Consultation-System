### 一、相关技术

- 前端：`jsp`、`HTML` / `CSS` / `Javascript`。
- 后端：`Spring`。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`

### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示

系统分为：用户、管理员/医生。

用户功能：1. 登录、2. 注册、3. 首页、4. 医生坐诊、5. 健康资讯、6. 新闻资讯、7. 我的信息、8. 客服

管理员功能：1. 登录、2. 个人中心、3. 科室管理、4. 医生管理、5. 医生坐诊管理、6. 用户资讯管理、7. 用户管理、8. 预约挂号管理、9. 用户病例管理、10. 健康资讯管理、11. 系统管理
